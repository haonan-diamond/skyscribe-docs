## Base URL

All API requests should be made to:

```
https://api.skyscribe.ai/v1
```

## Authentication

All API endpoints require authentication using API keys. Include your API key in the `Authorization` header of each request:

```bash
Authorization: Bearer YOUR_API_KEY
```

### Getting your API key

1. Sign up for a SkyScribe account
2. Navigate to your dashboard
3. Go to the API Keys section
4. Create a new API key or copy an existing one

<Warning>
Keep your API keys secure and never share them publicly. If you believe your key has been compromised, revoke it immediately and generate a new one.
</Warning>

## Rate Limits

API requests are subject to the following rate limits based on your plan:

| Plan | Requests per minute | Concurrent transcriptions |
|------|---------------------|---------------------------|
| Free | 10 | 2 |
| Pro | 60 | 10 |
| Enterprise | Custom | Custom |

If you exceed your rate limit, you'll receive a `429 Too Many Requests` response.

- M4A
- FLAC
- OGG
- AAC

**Video formats:**
- MP4
- MOV
- AVI
- MKV
- WEBM

**Maximum file size:** 500MB per file

## Response Format

All API responses are returned in JSON format. Successful responses include a `200 OK` status code. Errors include appropriate HTTP status codes and error messages:

```json
{
  "error": {
    "message": "Invalid API key",
    "code": "invalid_api_key",
    "status": 401
  }
}
```

## Common Error Codes

| Status Code | Description |
|-------------|-------------|
| 400 | Bad Request - Invalid parameters |
| 401 | Unauthorized - Invalid or missing API key |
| 404 | Not Found - Resource doesn't exist |
| 429 | Too Many Requests - Rate limit exceeded |
| 500 | Internal Server Error - Something went wrong on our end |
